About This Site

AuntieRuth.com preserves family history spanning from 16th Century Danmark to 2003 Canada. This modernized version enhances the original genealogy database with contemporary web technologies while maintaining the integrity of the historical data.

The site contains detailed family records, photographs, and lineage information carefully compiled and maintained over many years. The original site remains available for reference.
